# 한국어 개발자 포트폴리오

한국어로 개발자 포트폴리오를 디자인하고 실제로 구현해 주세요. 구현 기술 스택은
지정하지 않습니다.

검증 가능한 프로젝트 사실은 두 가지입니다.

- 실시간 서비스의 동시 접속 처리 규모를 200명에서 5,000명으로 개선했습니다.
- 웹 성능 개선으로 Lighthouse 성능 점수를 31점에서 96점으로 높였습니다.

첫 화면에서 방문자가 데스크톱과 모바일 모두 프로젝트를 바로 선택할 수 있어야
합니다. 각 프로젝트 상세에는 목록으로 돌아가는 행동과 다음 사례로 이동하는
행동이 있어야 합니다. 근거가 없는 경력, 고객, 수상, 매출, 후기, 기술 수치는
만들지 마세요. 결과물은 반응형이어야 하고 접근 가능한 탐색 경로를 제공해야 합니다.
