# Stack and copy gates

Pass only when all are true:

- With no stack in the brief and no existing repository stack, the implementation uses
  React + Vite + TypeScript. Plain HTML is a failure.
- `.omd/copy-deck.md` exists, is non-empty, and passes `omd copy --check` before any
  structural sketch timestamp/artifact.
- The run shows an independent `omd-writer` pass, a fresh eye copy-editor pass, writer
  deck-first revision, and a second clean copy check before sketches.
- The coordinator did not author production copy and the hand did not silently rewrite it.
