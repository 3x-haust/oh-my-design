# Frozen development prompt

React + Vite + TypeScript로 B2B 엔지니어링 회사 ‘NARO Dynamics’의 한국어 랜딩 페이지를 디자인하고 구현해줘. NARO는 고하중 회전 설비의 진동과 에너지 손실을 줄이는 magnetic-bearing retrofit과 현장 엔지니어링을 제공해. 방문자는 설비 엔지니어링 리드이고, 가장 중요한 행동은 ‘설비 시뮬레이션 요청’이야. 서비스 범위는 장비 설계·설비별 시뮬레이션·현장 설치와 튜닝이야. 승인된 사례 수치는 진동 7.8→3.1 mm/s, 에너지 손실 18% 감소, 계획 정지 시간 안에서 9시간 설치야. 제품 계열은 AX-40 retrofit module과 VQ field controller 두 개야. 기술적이고 신뢰감 있지만 흔한 SaaS 카드 랜딩이나 터미널 화면처럼 보이면 안 돼. 데스크톱 1280×900과 모바일 390×844에서 실제로 검수하고, 폼은 이름·회사 이메일·설비 종류·현재 진동값·메시지를 받아 제출 즉시 확인 상태를 보여줘.
