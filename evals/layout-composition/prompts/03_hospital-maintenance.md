# Frozen held-out prompt B

React + Vite + TypeScript로 병원 장비 유지보수 서비스 ‘CIRCA Care’의 한국어 랜딩 페이지를 디자인하고 구현해줘. 방문자는 병원 의공팀 리드고, 가장 중요한 행동은 ‘장비 위험도 진단 요청’이야. CIRCA는 예방 점검 일정, 부품 수명 예측, 현장 정비 이력을 한 흐름으로 관리해. 승인된 사실은 평균 미계획 정지 31% 감소, 점검 누락 42% 감소, 24시간 이내 위험도 보고서 제공이야. 차분하고 정확하며 신뢰감 있어야 하지만 흔한 SaaS 카드/그라데이션/대시보드 목업 랜딩처럼 보이면 안 돼. 데스크톱 1280×900과 모바일 390×844에서 실제 검수하고, 진단 폼은 병원명·업무 이메일·장비군·보유 대수·최근 고장 내용을 받은 뒤 제출 즉시 접수 상태와 다음 단계를 보여줘.
