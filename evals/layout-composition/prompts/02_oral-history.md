# Frozen held-out prompt A

React + Vite + TypeScript로 한국어 디지털 아카이브 ‘파장기록소’의 랜딩 페이지를 디자인하고 구현해줘. 파장기록소는 사라지는 한국 해안의 소리와 구술 기록을 보존하고 공개해. 방문자는 연구자, 창작자, 지역 주민이고 가장 중요한 행동은 ‘기록 듣기’야. 승인된 사실은 12년간 184개 해안 지점에서 3,720개 기록을 수집했고, 각 기록에 장소·날짜·기록자·전사문이 있다는 거야. 첫 화면에서 아카이브의 성격과 바로 들을 수 있다는 점이 분명해야 해. 에디토리얼하고 감각적이되 박물관 템플릿, 거대한 글자만 있는 화면, 똑같은 카드 목록처럼 보이면 안 돼. 데스크톱 1280×900과 모바일 390×844에서 실제 검수하고, 샘플 기록 재생 버튼은 클릭 즉시 재생 상태와 진행 상황을 보여줘.
